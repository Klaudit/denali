A robust Node.js framework
for building ambitious APIs.

 
 
 

Heads up!

Denali is still in alpha. We don't recommend using it for production apps just yet. Check out the road to beta to see what's left!

Data Flexibility

Thanks to it's unique model + adapter approach, Denali let's you use whatever ORM you want. And Denali Addons will work with it out of the box. You can even use multiple ORMs in the same project to fully leverage the strengths of each data store (a.k.a polyglot persistence).

Developer Happiness

Denali ships with a top-notch CLI which can run reload-on-change development server, scaffold new code, run tests, and more. Extensibility is baked into the core as well, with Addons providing a powerful, zero-configuration approach to sharing and leveraging community code.

API First

You won't find any templating languages, views, partials, etc. Denali is unapologetically API first. First class support for JSON serializers, RESTful routing, and powerful testing helpers make building APIs easier than ever.
