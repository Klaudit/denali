denali

This is the main module exported by Denali when it is loaded via require/import. This exports convenient shortcuts to other modules within Denali. Rather than having to import Addon from 'denali/dist/lib/runtime/addon', you can just import { Addon } from 'denali'.

Exports

Serializer

Serializers are responsible for determining what data gets sent over the wire, and how that data is rendered into a JSON response. Check out the guides for details.

Errors

An errors module based on http-errors. Useful for standardizing how you handle error responses. Check out the guides or the http-errors docs for details.
