# Denali

An opinionated Node.js framework for building JSON APIs.

```sh
$ npm install -g denali
$ denali new todo

$ cd todo && denali server
bookshelf server up and serving on port 3000

$ curl localhost:3000
{
  denaliVersion: "{{project.version}}",
  message: "Welcome to Denali!"
}
```
