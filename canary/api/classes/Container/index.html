    Container

    Defined in: lib/runtime/container.js Line: 9
    Module: denali.runtime

    The Container houses all the various classes that makeup a Denali app’s runtime. It holds references to the modules themselves, as well as managing lookup logic (i.e. some types of classes fall back to a generic “application” class if a more specific one is not found.
