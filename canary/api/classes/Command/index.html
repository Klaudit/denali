    params
    Defined in: lib/cli/lib/command.js Line: 31

    Description

    An array of positional paramters to this command. For example,

    params: [ 'foo', 'bar' ]

    When run with:

    $ denali mycommand Hello World

    Would result in:

    run(params) {
  params.foo // "Hello"
  params.bar // "World"
}
