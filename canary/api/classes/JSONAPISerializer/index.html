    Defined in: lib/data/serializers/json-api.js Line: 500

    Description

    Unlike the other serializers, the default parse implementation does modify the incoming payload. It converts the default dasherized attribute names into camelCase.

    The parse method here retains the JSONAPI document structure (i.e. data, included, links, meta, etc), only modifying resource objects in place.

    Parameters
    payload
    :Object

    the JSONAPI document to parse
