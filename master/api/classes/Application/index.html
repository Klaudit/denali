generateConfig()
Private
Defined in: lib/runtime/application.js Line: 89

Description
Take the loaded environment config functions, and execute them. Application config is executed first, and the returned config object is handed off to the addon config files, which add their configuration by mutating that same object. The resulting final config is stored at `application.config`, and is registered in the container under `config:environment`.
