denali

A Node.js framework for building robust APIs

$ npm install -g denali
$ denali new my-api
$ cd my-api && denali serve

What is Denali?

Denali is a framework that makes building robust JSON APIs simple. It borrows the best concepts from frameworks like Rails, is built on the rock solid foundation of Express.js, and is designed for developer happiness.

API-centric

You won't find any templating languages, views, or other HTML rendering support out of the box. Denali is unapologetically focused on APIs first. First class support for JSON serializers, ORM agnostic data adapters, and RESTful routing make APIs easier than ever.

Developer happiness

Denali ships with a top-notch CLI which can run reload-on-change development server, scaffold new code, run tests, and even deploy your app. Extensibility is baked into the core as well, with Engines providing a powerful approach to incorporating third party code.
